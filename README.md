# Encuesta de Satisfacción - Daniel Soledad

Este proyecto consiste en la creación de una encuesta de satisfacción utilizando: **HTML**, **CSS**, **JS Vanilla**, para cumplir con el examen práctico del Módulo 1 del Máster de Profesionalidad **_Desarrollo de Aplicaciones con Tecnologías Web_**.

## Initial Commit

- .gitignore
- README.md
- editorconfig
- index.html

## Images Folder

Lo usaremos para conseguir el favicon y las imágenes que usaremos en el componente

## Styles Folder

Este directorio contendrá los 2 archivos CSS que usaremos para dotar de estilos a nuestro proyecto. El reset se encargará de que los elementos no cuenten con herencia, así nuestros estilos definidos en _index.css_ se aplicarán sin inconvenientes.

## Scripts folder

En este directorio se añadirá el script JS para otorgarle la funcionalidad de recoger el dato de un botón para luego enseñarlo en otro lugar
